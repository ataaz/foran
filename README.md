# Draftly Full Website Builder Export

This export includes a frontend + backend starter so users can continue building a fully functional website after the 3D preview stage.

## Structure

- frontend/ -> generated website UI and assets
- backend/ -> Express API starter (contact + leads + health)

## Run locally

1. Backend
   - cd backend
   - npm install
   - npm run dev
2. Frontend
   - serve frontend (or any static server)

## Notes

- Existing 3D flow assets are preserved.
- Extend backend routes for auth, CMS, payments, and storage as needed.
